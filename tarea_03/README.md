# Comentarios sobre el proceso

Al hacer la página web, seguí las instrucciones que trabajamos en la clase, crear la carpeta doc y activar Github Pages, con todo disponible y público.

Con ese conocimiento también agregué las primeras etiquetas como <html> <head> y <body>. A medida que iba escribiendo, me imaginaba una página con el contenido centrado, cosa que no lo tenía previsto en el diagrama que hice inicialmente. Busqué tutoriales para poder encontrar las intrucciones y las etiquetas.

No recordaba cómo agregar las imágenes, por lo que en lo que más trabajé fue en ese aspecto. Sabía que era con la etiqueta <img> pero no sabía cómo incluir la imagen, así que Busqué un tutorial para guiarme. Al inicio, solo iba a agregar la imagen del gráfico, pero se veía muy vacío. Agregué una imagen con portadas de 4 K-Dramas que, sin duda, tienen una popularidad grande.

Incorporé varios <div> para agrupar los contenidos, y agregué etiquetas <a> para los enlaces que fueron añadidos donde se mencionan Netflix, Viki Prime Video y Disney+, permitiendo que se pueda acceder a las páginas de las plataformas. Al final de la página también añadí un link a Wiki Drama, invitando a explorar más K-Dramas.
